# Tor Browser

Privacy and anonymity browser based on Firefox.

## Features

- Tor network integration for anonymous browsing
- Privacy-focused defaults
- NoScript and other privacy extensions included
- Multi-language font support

## Installation

1. Download from the official Tor Project website
2. Run `Browser/tor-browser.exe`

## Usage

Launch the browser and it will automatically connect to the Tor network for anonymous browsing.
